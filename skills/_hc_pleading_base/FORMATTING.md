# FORMATTING.md — Indian High Court formatting specification

Plain-language formatting reference for `indian-hc-drafting`. Any advocate can apply these settings in Microsoft Word, LibreOffice, Google Docs, or any word processor. The bench-config.md in the user's case folder overrides any value here.

---

## Paper & layout

- **Paper size:** A4 (most HCs · default)
  - Some Original Side practices use Legal — verify against your bench-config
- **Print:** one-side preferred · two-side only where Registry expressly permits

## Typography

- **Font family:** Times New Roman (most HCs · default)
  - Advocates commonly use Arial / Calibri / Cambria — all accepted in most HCs as long as serif legibility is maintained
- **Font size body:** 14 point (preferred for most HCs)
  - 12 point acceptable for some State Manuals (e.g., Bombay Civil Manual at District level)
- **Font size headings:** 14 point bold
- **Line spacing:**
  - **1.5** — standard for most HCs (Bombay, Karnataka, Madras, Calcutta, Allahabad, Kerala, Gujarat, etc.)
  - **2.0 (DOUBLE)** — mandatory for **Delhi HC** (Original Side Rules 2018, Rule 1) and **Punjab & Haryana HC** (Rules & Orders Vol V)

## Margins

- **Left (binding side):** 4 cm — mandatory across all HCs
- **Top:** 2.5 cm
- **Bottom:** 2.5 cm
- **Right:** 2.5 cm

## Page numbers

- Centred at bottom of page
- Continuous pagination from main pleading through accompanying applications and annexures

## Section headers

- **Bombay HC (all benches):** spaced — `F A C T S`, `G R O U N D S`, `S Y N O P S I S`, `I N D E X`
- **Most other HCs:** Title Case bold — `Facts`, `Grounds`, `Synopsis`, `Index`
- Per `bench-config.md` Section 4

## Cause Title block

- Court header — per `bench-config.md` Section 1
- Case number line — per `bench-config.md` Section 3
- Parties block format — per `bench-config.md` Section 3 (parties_separator)

## Annexure marker

- Inline marker in Facts — `bench-config.md` Section 6 (`annexure_prefix`)
  - Bombay HC: `ANNEXURE-A, B, C, ...`
  - Punjab & Haryana HC: `Annexure P/1, P/2, ...` (petitioner) / `Annexure R/1, R/2, ...` (respondent)
  - Karnataka HC: `Annexure-A, B, C, D, E, F, G, H, J, K, ...` (letter I **omitted** per Writ Proceedings Rules 1977)
  - Kerala HC: `Exhibit P1, P2, ...` (default) or `Annexure-A, B, ...`
  - Madras HC OS: `Document No. 1, 2, 3, ...`
- Consolidated `LIST OF ANNEXURES` table at end of pleading

## Verification block

- Per CPC Order VI Rule 15 — verbatim
- Layout (advocate name, designation, place, date) per bench-config Section 7 (`counsel_block_template`)

## Accompanying applications

- Each accompanying application repeats its own Cause Title in the bench's idiom
- Own facts / own prayer / own counsel block / own affidavit
- Page-numbered continuously with the main pleading

## Bench-specific exceptions to flag in your bench-config

- **Delhi HC OS Rules 2018:** advocate name + Bar Council enrolment + phone + email **mandatory** at foot of every pleading
- **Punjab & Haryana HC:** watermarked plain paper or "clearly legible properly-spaced computer prints"
- **Gujarat HC Rules 1993:** foolscap paper with 2-inch margin (A4 with the standard margin block above is acceptable in current practice — verify Registry)
- **Kerala HC Rules 1971:** foolscap, 4cm outer + 1.5cm inner margins, book-stitched (A4 acceptable in current practice — verify Registry)

---

## How to apply these in Microsoft Word / LibreOffice

1. **Page setup:** Layout → Margins → Custom: left 4 cm, top/bottom/right 2.5 cm; Paper size: A4
2. **Body style:** Home → Styles → Normal → Modify → Font: Times New Roman 14 pt; Line spacing: 1.5 lines (or 2.0 for Delhi/P&H)
3. **Heading styles:** Heading 1 / 2 / 3 → Modify → Font: Times New Roman 14 pt bold
4. **Page numbers:** Insert → Page Number → Bottom of page → centred
